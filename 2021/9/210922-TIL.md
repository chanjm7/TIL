## Java
- Object 클래스 메서드
	- equals()
		- 객체 자신과 주어진 객체를 비교하는 메서드
	- hashCode()
		- 객체의 해시코드를 반환하는 메서드
	- toString()
		- 객체에 대한 정보를 문자열로 제공하기 위한 메서드
	- clone()
		- 자신을 복제하여 새로운 인스턴스를 생성하는 메서드
	- getClass()
		- 자신이 속한 클래스의 Class객체를 반환하는 메서드
